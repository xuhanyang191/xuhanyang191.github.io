## 🎮 操作方式

### 键盘操作

| 游戏 | 操作 |
|------|------|
| 贪吃蛇/2048 | ⬆⬇⬅➡ 方向键 |
| 打砖块/乒乓球 | ⬅➡ 移动 |
| 飞机射击 | ⬅➡ 移动，空格/⬆ 射击 |
| 扫雷 | 点击翻开，右键标旗 |
| 打地鼠 | 点击地鼠 |
| 俄罗斯方块 | ⬅➡ 移动，⬆ 旋转，⬇ 加速，空格 直接落下 |
| **3D 跑酷** | ⬆/空格 跳（可二连跳），⬇ 下蹲，⬅➡ 换道 |

**跑酷快捷键：** `L` 自动模式 · `C` 切换皮肤 · `M` 开关音效

### 手机操作
所有游戏都支持触屏，底部有虚拟按键。
